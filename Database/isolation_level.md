# Isolation level

격리 레벨(isolation level)은 트랜잭션에서 일관성이 없는 데이터를 허용하도록 하는 수준을 의미

### Isolation Level

1. Read Uncommitted(level 0)
2. Read Committed(level 1)
3. Repeatable Read(level 2)
4. Serializable(level 3)

총 4 가지 격리수준으로 나누어 사용한다.
